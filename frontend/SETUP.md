# راه‌اندازی Frontend

## پیش‌نیازها

قبل از شروع، مطمئن شوید که Node.js و npm نصب شده‌اند:

```bash
# بررسی نسخه Node.js (باید 18 یا بالاتر باشد)
node --version

# بررسی نسخه npm
npm --version
```

اگر نصب نیست، از [nodejs.org](https://nodejs.org/) دانلود کنید.

## نصب وابستگی‌ها

```bash
cd frontend
npm install
```

این دستور تمام وابستگی‌های مورد نیاز از جمله Next.js را نصب می‌کند.

## اجرای پروژه

```bash
npm run dev
```

پس از نصب، خطای TypeScript درباره `next/link برطرف می‌شود.
